\subsection{Update Interface}
Roadmap:
\begin{itemize}
\item Ablauf beschreiben
\item JSON 
\item Data-Binding 
\item LocalStorage
\item Submit
\end{itemize}
Ein PaaS-Anbieter wird in der Wissensbasis als ein \acs{JSON}\footnote{\label{foot:json}http://json.org}-Eintrag gespeichert. JSON basiert auf zwei Datenstrukturen \ref{foot:json}: 
\begin{itemize}
\item \textit{Schlüssel/Wert-Paar} als ein Objekt in JSON.
\item \textit{Geordnete Liste von Werten} oder als ein Array bekannt.
\end{itemize}
